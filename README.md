# 🏧 ATM Machine Simulation

A Python-based ATM simulation that allows users to authenticate using a PIN and perform basic banking operations such as balance inquiry, deposits, and withdrawals.

## ✨ Features

* 🔐 PIN Authentication
* 💰 Balance Inquiry
* ➕ Deposit Money
* ➖ Withdraw Money
* ⚠️ Insufficient Balance Handling
* 🚪 Exit Functionality

## 🛠️ Tech Stack

* Python 3

## 🚀 How to Run

```bash
python atm_machine.py
```
